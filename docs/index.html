greedy’s documentation¶

greedy a Python package which brings topological (greedy) coloring to GeoPandas. Several coloring strategies all accessible using one line of code:

gdf['colors'] = greedy(gdf)

Install greedy using conda from conda-forge:

conda install -c conda-forge greedy

Or from PyPI using pip:

pip install greedy
